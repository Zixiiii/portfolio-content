Property price forecast

In this project, I used Python to construct and train regression models between sales prices and other property characteristics by extracting information from data provided by the CCAO(Cook County Assessor's Office, Illinois) that contains over 500,000 property sale records. I eventually applied the model, using known features to predict unknown sales prices.
